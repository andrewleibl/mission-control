## Overview

This package contains all website copy, policies, product content, and conversion elements needed to take clutchbarbersupply.com from its current state (audit-grade ~1% CVR) to a professional, trust-optimized ecommerce site capable of 3%+ conversion rates.

All content is:
- Written for **professional barbers**, not casual shoppers
- Tone: **knowledgeable, direct, community-focused**
- Conversion-focused: based on the findings in the March 2026 conversion audit
- Shopify-compatible (all policy pages, collections, and tags structured for direct import)

### 4. `refund-policy.md`
Professional 30-day return policy modeled after top Shopify stores (Zappos standard), including:
- 30-day returns on unused/unopened items
- Defective item process (free prepaid return label, exchange or store credit)
- Opened/used defective item path
- Non-returnable items list (events, final sale)
- Exchange process
- Refund timeline (5–7 business days)
- Contact: returns@clutchbarbersupply.com

**Key decisions:** Zero hostile language — completely rewrites the current policy which contains "all sales are final," "buyer's remorse will not be accepted," and a Gmail contact. This policy change alone is estimated to recover 0.5–1% CVR based on Baymard Institute data.
