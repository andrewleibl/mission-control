## Day 1: Foundations

### What We Do (Tell Them This)
We generate qualified appointments for landscaping companies using Facebook ads. We call every lead ourselves before passing them to the client — so the client only talks to people with a budget, ready to hire.

Our job is to get landscaping business owners on a 15-minute call with Andrew.

### The Offer to Know Cold
- **30-day package:** $1,000 for 10 guaranteed qualified appointments
- **90-day package:** $2,500 for 30 guaranteed qualified appointments
- Shortfall guarantee: $100 refund per missed appointment

### Who We're Calling
- Owners/operators of landscaping companies (mowing, cleanups, installs)
- Mid-sized businesses — not solo owner-operators, not huge commercial operations
- Pulled city by city from public business directories
- Target: owners who want to grow, not maxed-out single operators

### What You're NOT Doing
- You're not closing the deal — Andrew does that
- You're not explaining everything — just spark interest, book the call
- You're not leaving voicemails every time — only on 2nd and 4th attempts

## Day 2: Script Walkthrough

Read `cold-caller-script.md` together line by line.

**Checkpoints:**
- [ ] Caller can recite the opening from memory
- [ ] Caller understands the 5-touch call cadence
- [ ] Caller knows all 6 major objections and the response approach
- [ ] Caller knows voicemail script
- [ ] Caller knows what to log in GHL

Shadow calls (Andrew dials, caller listens):
- Minimum 10 calls observed before first live attempt

---

## Day 3: Role Play

**Andrew plays the prospect. Caller runs the script.**

Run through each scenario:
1. Prospect says YES immediately
2. Prospect says "not interested"
3. Prospect asks about price
4. Prospect says "send me an email"
5. Prospect says "I use another company"
6. Prospect says "bad time, call back later"
7. Prospect hangs up

**Pass criteria:** Caller handles all 7 without freezing or going off-script.

Repeat until smooth — this can be done via phone or in-person.

---

## Day 4–5: Live Calls

- Andrew stays on standby (Telegram or WhatsApp) for first live sessions
- Caller messages Andrew before dialing a new batch: "Starting [X] calls — [City] list"
- If a call goes sideways, caller marks it in GHL and texts Andrew immediately
- End-of-session debrief: review 3 best calls, 3 worst calls

## Daily Process (Once Live)

### Morning
1. Pull today's list from [wherever Andrew stores it] — [update this with actual source]
2. Load numbers into dialer / SMS platform
3. Set GHL tasks for any callbacks due today
4. Start dialing by 9:00 AM

### During
- Log every call disposition immediately after hanging up
- Set next-task date for every non-dead lead
- Text Andrew if you get a hot interested prospect in real time

### End of Day
Send Andrew a recap via [Telegram/WhatsApp/GHL]:
```
Today's recap:
Dials: X
Live answers: X
Interested: X
Booked: X
Voicemails: X
DNC/dead: X
Top call of the day: [brief note]
Questions/stuck points: [anything]
```

## GHL Dispositions to Use

| Label | When to Use |
|-------|-------------|
| No Answer | Rang, no response |
| VM Left | Left voicemail |
| Answered - Not Interested | They said no or hung up |
| Answered - Interested | Wants more info or call |
| Booked | Call scheduled |
| Dead | 5 attempts, no response |
| DNC | Asked to be removed |
| Bad Number | Disconnected / wrong number |

## Quality Standards

**You'll be reviewed on:**
- Dial volume (80–100/day target)
- Booking rate (1–2 booked calls/day target)
- GHL accuracy (every call logged with notes)
- Response time to Andrew's messages (within 30 min during session)

**You won't be penalized for:**
- Rejections (expected — this is cold calling)
- Objections you couldn't overcome (that's training, not failure)
- Asking questions (always better than guessing)

---

## Escalation: When to Contact Andrew Immediately

- You get a genuinely interested prospect who asks for Andrew directly
- Someone gets aggressive or threatens legal action
- You accidentally dial the same person twice on the same day
- A booked call cancels — update GHL AND text Andrew
