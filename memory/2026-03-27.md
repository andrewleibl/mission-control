### Workflow Pattern Established

**For Weekly Client Looms:**
1. Go to Tasks → Retention view
2. Click Friday "Weekly Report" event
3. Click "Open for Loom" button
4. Land on `/loom-recording` page with:
   - Charts showing week's performance
   - The Big 3 questions pre-filled
   - Edit any section by clicking pencil
   - Download JSON report (top-right icon)
5. Record Loom narrating the dashboard
6. Send download to client

### Design Decisions

**Red Glow Theme:**
- Background: `linear-gradient(180deg, #1A1A1A 0%, rgba(229, 62, 62, 0.02) 100%)`
- Radial glow overlay: `radial-gradient(ellipse at center, rgba(229, 62, 62, 0.06)...)`
- Section cards: `border: 1px solid rgba(229, 62, 62, 0.1)` with subtle gradient
- Matches retention calendar aesthetic

**Unified Edit Mode:**
- One edit button per section (not per field)
- Opens full editing interface for:
  - Main summary textarea
  - All list items with individual inputs
  - Add/remove item controls
  - Section-level save/cancel
- Cleaner UX than field-by-field editing

**Arrow Bullet Style:**
- Red arrow (→) prefix for all list items
- Consistent with user's preferred formatting
- High contrast against dark background

### Key Learnings (User-Validated as "Big Win")

1. **Unified Edit > Granular Edit** — User explicitly wanted ONE edit button to control entire section content, not individual field editing

2. **Visual Hierarchy > Information Density** — Clean separation between charts (top) and narrative sections (bottom). Numbered badges (1,2,3) create clear progression.

3. **Action-Oriented Layout** — Edit button placement (top-right of section) follows standard UX patterns. Download icon (not text) keeps header clean.

4. **Consistent Theming** — Red glow/styling creates visual continuity with retention calendar. User requested matching aesthetic.

5. **Iterative Refinement Works** — Built basic version first, then adjusted glow levels, removed "needs attention" badge, unified edit mode based on feedback. User satisfaction came from successive refinements.
